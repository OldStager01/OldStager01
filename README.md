### Projects

**Prism** — a cloud platform that lets a healthcare provider's patients pull up their own procedure records instantly, instead of collecting them on physical discs across a counter. Built it solo for Josh
Software's company-wide engineering bounty, which it went on to win. Now running in production. Django, Next.js, AWS, Celery, Postgres. *(Private client, so internals stay out of this.)*

**Samadhan** — on-prem agentic AI for a workflow inside a regulated bank. Instead of one big prompt, a team of agents breaks the task into steps and grounds each one in the actual official guidance, so the output holds up. Went to the iDEA Hackathon finals — top 40 out of 1,600 teams. Go, Python, LangGraph, Kafka, Next.js. *(Problem statement is under wraps.)*

**[Cloud Autoscaler](https://github.com/OldStager01/cloud-autoscaler)** — real-time cluster autoscaling in Go. Each cluster gets its own goroutine pipeline; the metrics collector sits behind a circuit breaker so a flaky endpoint doesn't wreck scaling decisions, and scale-up/scale-down have separate cooldowns. Live dashboard over WebSockets. Go, TimescaleDB, Docker.

**[Fintech Fraud Detection](https://github.com/OldStager01/fintech-fraud-detection)** — transaction monitoring with live risk scoring. React + Vite front end, Rails API, Postgres.

**[Flicksta](https://github.com/OldStager01/flicksta)** — a curated Flickr image platform handling 50K+ images, server-rendered with Django, Redis and HTMX on AWS.

**[Tree Plantation](https://github.com/OldStager01/Tree-Plantation)** — a platform connecting people and NGOs for tree planting. Role-based access, an Express API, Firebase auth, some Gemini-powered bits.

More over at [@OldStager01](https://github.com/OldStager01?tab=repositories).
